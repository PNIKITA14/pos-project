# 🛒 SmartMart POS System

## 📌 Project Description
SmartMart POS is a full-stack Point of Sale (POS) system built using React, Node.js, Express, and MongoDB.  
It allows users to browse products, add items to a cart, calculate billing with GST, and store orders in a database.

---

## 🚀 Features
- 🛍️ Product listing from database
- ➕ Add to cart functionality
- 🔄 Increase/decrease quantity
- ❌ Remove items from cart
- 💰 Billing system with GST (5%)
- 📦 Order storage in MongoDB
- 📊 Dashboard with total sales and orders
- 🔐 Login & Logout system (using localStorage)

---

## 🧑‍💻 Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
